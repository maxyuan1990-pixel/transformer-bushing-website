# Transformer Bushing Export Website

Astro single-page export website for transformer bushings.

## Run locally

```powershell
npm install
npm run dev
```

Then open the local URL printed by Astro, usually `http://127.0.0.1:4321`.

## Build

```powershell
npm run build
```

The production files will be generated in `dist/`.

## Edit content

- Main page content: `src/pages/index.astro`
- Global styles: `src/styles/global.css`
- Local visual assets: `public/assets/`
- Favicon: `public/favicon.svg`

Replace `VoltGate`, `sales@example.com`, and the demo Wikimedia image URLs with your real company name, email address, and product photos before publishing.
